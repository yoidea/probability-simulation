## 内容

中学2年生で学ぶ確率の問題を、乱数を使ったシミュレーションで体験できます。

試行回数を増やしていくことで、実験結果の割合が理論確率に近づいていく様子をグラフで確認できます。授業などでご自由にお使いいただけます。

### コイン投げ

コインを何度も投げて、表が出る割合がどのように変化するかを観察します。  
理論確率：1/2

### サイコロ振り

1つのサイコロを何度も振って、4の目が出る割合がどのように変化するかを観察します。  
理論確率：1/6

### トランプ引き

52枚のトランプから1枚引いて、ハートが出る割合がどのように変化するかを観察します。  
理論確率：13/52 = 1/4

### 玉取り出し

赤玉2個、白玉1個、青玉3個が入っている袋から1個取り出して、赤玉が出る割合がどのように変化するかを観察します。  
理論確率：2/6 = 1/3

## 実装

すべて **HTML + CSS + JavaScript の単一ファイル**で構成されています。  
外部ライブラリは使用していません。
